What do we mean by clustering? We aim to partition all points into $K$ classes $C_1, C_2, \cdots, C_K$ and find $K$ centers $\mu_1, \mu_2, \cdots, \mu_k$, such that

$$\sum_{k}\sum_{i} r_{ik} \|x_i - \mu_k\|^2$$

where $r_{ik}$ is the indicator of where $x_i$ is in class $C_k$.

K-Means Clustering¶

Given $K$, K-Means algorithm works as follows,

  1. Randomly initialize $K$ centers $\mu_1, \mu_2, \cdots, \mu_k$
  2. Iteratively assign each $x_i$ to its nearest center $\mu_k$, and then reestimate $\mu_k$ based on assignment until converge.

Exercise 0: First, we will implement the assign function, which assigns data points to its nearest center. For example, suppose data = [[101, 0], [99,1], [-100,2]], mu = [[100,0], [-100, 0]], then you would expect to return membership = [0, 0, 1]. If there are multiple centers that are equidistant to a point, pick the center with the lowest index, i.e., the first center in mu that is nearest to the point. (3 points)

def calculate_euclidean_distance (point_a, point_b):
    diffs = np.subtract(point_a, point_b)
    square_diff = [diff**2 for diff in diffs]
    return np.sqrt(np.sum(square_diff))

def assign(data, mu):
    # data: (N, 2) numpy.array
    # mu: (K, 2) numpy.array
    # return: (N,) numpy.array indicating class membership of each data point
    
    membership = np.zeros(len(data))
    ###
    for i, data_point in enumerate(data):
        distances_per_datapoint = []
        for center in mu:
            distances_per_datapoint.append(calculate_euclidean_distance(data_point, center))
        
        min_distance = min(distances_per_datapoint)
        
        membership[i] = distances_per_datapoint.index(min_distance)
        
    ###
    return membership

def re_estimate(data, membership):
     # data: (N, 2) numpy.array
     # membership: (N,) numpy.array indicating class membership of each data point   
     # mu: (K, 2) numpy.array
             
    mu = None
    ###
    
    # K is the number of classes
    K = len(set(membership))
    
    new_mus = []
    for i in range(K):
        
        # Find all points in data that were assigned to cluster i & based on these points,
        # Find the new centroid by taking the average of the points that were assigned to cluster_i
        points_prev_assinged_to_i = [d for d, m in zip(data, membership) if m == i]
        
        # Ensure we do not have an empty vector
        if points_prev_assinged_to_i:
            child_mu = []
            # Average all of th 1st position of the coordinate through nth position of a data point
            for j in range(len(points_prev_assinged_to_i[0])):
                points_to_avg = [point[j] for point in points_prev_assinged_to_i]
                child_mu.append(np.mean(points_to_avg))
            print(child_mu)
            
            new_mus.append(child_mu)
                    
    mu = np.array(new_mus)
    ###
    return mu

Kernel Density Estimation¶

We turn our attention to another question, what is the probability distribution of data? A powerful method is kernel density estimation (KDE). The probability density function is given by $$ f(x) = \sum_{i=1}^N \frac{1}{N}K(\frac{x-x_i}{h})$$ where $K(z)=\frac{1}{\sqrt{2\pi}} \exp(-\frac{\|z\|^2}{2})$ is a kernel function and $\|z\|^2$ is the squared euclidean norm). def kernel(z):
    # z: (N, 2) numpy.array
    # returns (N, 1) numpy.array
    
    ###
    # Calculate the 
    euc_norm = []
    for Z in z:
        euc_norm.append(la.norm(Z, 2, keepdims=False))

    euc_norm_sq = [norm**2 for norm in euc_norm]
    x = [-1*(norm_sq)/2 for norm_sq in euc_norm_sq]
    
    K_numer = [np.exp(X) for X in x]
    K_denom = np.sqrt(2*np.pi)
    
    return np.array([num/K_denom for num in K_numer])
